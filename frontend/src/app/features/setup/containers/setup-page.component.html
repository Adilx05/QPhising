Platform Setup Required

Setup tamamlanmadan uygulama modülleri kullanılamaz. Lütfen backend setup adımlarını tamamlayın ve ardından sayfayı yenileyin.

  1. POST /api/setup/validate-db ile veritabanı doğrulamasını yapın.
  2. POST /api/setup/validate-sso ile SSO doğrulamasını yapın.
  3. POST /api/setup/finalize çağrısını tamamlayın.
